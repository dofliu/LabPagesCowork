隱私權聲明

本網站由國立勤益科技大學 智慧自動化工程系 劉瑞弘副教授實驗室（以下簡稱「本實驗室」）維護。 本隱私權政策說明我們如何收集、使用及保護您的個人資訊。

資料收集

本網站可能收集以下類型的資訊：

  • 瀏覽資訊：透過 GitHub Pages 伺服器日誌記錄的 IP 位址、瀏覽器類型、造訪時間等。
  • 主動提供的資訊：當您透過電子郵件聯繫我們時，您提供的姓名與電子郵件地址。

資料使用

我們收集的資訊僅用於以下目的：

  • 改善網站使用體驗
  • 回覆您的諮詢與合作需求
  • 進行網站流量分析（匿名彙總資料）

第三方服務

本網站使用以下第三方服務：

  • GitHub Pages — 網站代管服務（遵循 GitHub 隱私權政策）
  • Google Fonts — 字型服務
  • Font Awesome — 圖示字型服務

Cookie 政策

本網站本身不使用 Cookie。但上述第三方服務可能透過其 CDN 使用技術性 Cookie。

資料保護

我們採取合理的技術與管理措施來保護您的個人資訊，防止未經授權的存取、使用或洩露。 本網站透過 HTTPS 加密傳輸所有資料。

您的權利

您有權要求我們提供、修改或刪除您的個人資訊。 如有任何隱私權相關問題，請透過 dofliu@gm.ncut.edu.tw 聯繫我們。

政策更新

本隱私權政策最後更新日期為 2026 年 2 月。 我們可能不定期更新本政策，更新後的版本將直接發佈在本頁面。

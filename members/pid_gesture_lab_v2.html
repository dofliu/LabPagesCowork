📷 相機權限解決方案

手機瀏覽器需要 HTTPS 才能使用相機。本地 file:// 無法取得權限。

方案 A — 最快
不用相機，直接觸控拖動！
關掉這個提示，直接在下方極零點圖上用手指拖動 × 極點，效果完全一樣。
方案 B — 手勢控制
把這個檔案放到網路上，用 HTTPS 開啟：
① 到 github.com 開一個 repo
② 上傳此 HTML 檔案
③ 到 Settings → Pages → 啟用 GitHub Pages
④ 用手機開啟產生的 https:// 網址
⏱ 約需 5 分鐘設定，之後相機即可正常使用
方案 C — 電腦測試
在電腦上開終端機執行：
python3 -m http.server 8000
然後用瀏覽器開 http://localhost:8000
localhost 也算安全環境，相機可正常使用。

# LabPagesCowork 發文流程指南

## 發文頻率
- **目標**：每週至少2篇
- **時間**：週二、週五發文
- **主題**：研究相關優先

## 發文主題優先級
1. **RAG系統 / 檢索增強生成**
2. **風電運維 / SCADA資料分析**
3. **智慧監控平台 / AR + IoT**
4. **AI Agents / LLM自動化 / MCP**
5. **PLC程式自動生成**
6. **一般AI新聞（與研究相關）**

## 發文步驟
1. **蒐集素材** → 優先研究相關主題
2. **寫Markdown** → 存 posts/ 資料夾
3. **轉HTML** → 使用轉換工具
4. **Git操作** → add + commit + push
5. **建立PR** → 合併到主分支
6. **網站更新** → 確認網站顯示

## 檔案命名規則
```
YYYY-MM-DD-主題-簡稱.md
範例：2026-03-17-RAG系統進展.md
```

AgenticOCR：視覺 RAG 的第三塊積木！上海 AI Lab 讓 OCR 超越人類專家

2026年3月3日 RAG, 視覺語言模型, OCR, Agent

當 RAG（Retrieval-Augmented Generation）從純文字走向金融財報、技術手冊等視覺豐富的文件時，一個巨大的瓶頸出現了：我們真的需要把整頁 PDF 都丟給 LLM 嗎？大量的無關資訊不僅會稀釋 LLM 的注意力，更會在有限的視覺 token 預算下增加幻覺風險。為了解決這個問題，上海 AI 實驗室聯合北京大學、商湯科技在 2026 年 2 月 27 日發表了一篇名為 AgenticOCR 的論文，提出了一個革命性的概念：將 OCR 從靜態的預處理步驟，轉變為由查詢驅動的「智能代理」過程。

重點摘要

  • AgenticOCR 概念： 提出「智能 OCR」概念，不再是「全部解析」，而是「只解析你需要的」，將 OCR 變成一個動態、查詢驅動的代理過程。
  • 超越人類專家： AgenticOCR-8B 模型在 MMLongBench-Doc 基準測試上達到 66.4% 的準確率，超越了人類專家的 65.8%。
  • 視覺 RAG 的第三塊積木： AgenticOCR 可作為一個即插即用的中間件，無縫整合到現有的 RAG 流程中，扮演與 Embedding 和 Reranking 模組同等重要的角色。
  • 開源貢獻： 相關程式碼、模型和數據集均已在 GitHub 上開源，推動社群發展。

當「全部解析」遇上瓶頸

傳統的視覺 RAG 流程通常以「頁」為單位進行切割和檢索，這雖然直觀，但卻非常粗糙。一整頁的 PDF 可能包含了頁首、頁尾、無關的圖表和裝飾性元素。將這些充滿「視覺噪音」的資訊全部餵給生成模型，就像讓一個速讀專家去讀一本混雜了大量廣告的書，效率低下且容易出錯。

更致命的是，高解析度的文件頁面必須被壓縮成有限的視覺 token，這不可避免地會犧牲掉許多細節，例如小字號的註解、旋轉過的表格或複雜的公式，從而大大增加了模型產生幻覺的風險。AgenticOCR 的出現，正是為了解決這個「資訊粒度不匹配」的核心矛盾。

AgenticOCR：像人一樣「看」文件

AgenticOCR 的核心思想是讓模型學會像人一樣閱讀文件——不是逐字逐句地通讀，而是根據問題快速定位到相關區域，然後再進行精細閱讀。它透過一個統一的視覺互動工具 image_zoom_and_ocr_tool 來實現這個目標，這個工具整合了區域定位、幾何校正和內容識別等多種功能。

模型可以自主決定：

  • 看哪裡 (Where to look): 透過 layout 分析，定位到與查詢最相關的區塊。
  • 怎麼看 (How to orient): 對於旋轉的表格或圖片，可以先進行角度校正再識別。
  • 看多細 (What semantic granularity): 可以選擇只識別文字、表格、圖片或公式，實現按需解析。

這個過程被稱為「按需解壓縮視覺 token」（on-demand decompression of visual tokens），它有效地將檢索的粒度從死板的頁面級別解放出來，大大提升了信噪比。

圖一：AgenticOCR 作為視覺 RAG 中間件的工作流程。它接收檢索到的頁面，按需解析出關鍵證據，再提交給生成模型。

驚人的實驗結果：超越人類專家

AgenticOCR 的效果非常顯著。在極具挑戰性的長文件理解基準測試 MMLongBench-Doc 上，AgenticOCR-8B 模型取得了 66.4% 的驚人準確率，不僅超越了 Gemini-2.5-Pro (63.3%) 和 Claude-4-Sonnet (56.0%) 等頂級商業模型，甚至超過了人類專家的表現 (65.8%)。

在金融領域的 FinRAGBench-V 測試中，AgenticOCR 同樣以 78.6% 的準確率大幅領先所有先前的 Agentic 框架，證明了其在處理版面密集的複雜視覺文件時的優越性。

表一：AgenticOCR 在 MMLongBench-Doc 和 FinRAGBench-V 上的主要表現
模型 MMLongBench-Doc (ALL) FinRAGBench-V (ALL)
Gemini-2.5-Pro (Vanilla) 58.1 54.9
Gemini-2.5-Pro (+OCR) 63.3 64.9

結論與未來展望

AgenticOCR 的提出，為視覺 RAG 領域帶來了一個全新的、極具潛力的發展方向。它將 OCR 從一個被動的預處理工具，變成了 RAG 流程中一個主動、智能的「證據過濾器」。這不僅僅是技術上的突破，更是思維範式的轉變。

研究團隊將 AgenticOCR 定位為視覺 RAG 技術棧中的「第三塊積木」，與現有的 Embedding 和 Reranking 模組相輔相成，共同構建更高效、更精準的下一代 RAG 系統。隨著模型的開源，我們有理由相信，一個由社群驅動、圍繞 AgenticOCR 的生態系統將會迅速成長，為解決真實世界的複雜文檔智能問題提供強大的新動能。
